# Event Management

Event Management is a web application that is designed in a way that the organization and management of special occasions becomes extremely simple and efficient to do. Using this application, the event creator can create an event, invite attendees, give assignments, and check the progress. As for the attendees, they can check the event, RSVP, and know the complete address including the date and even time. The event organizers are namely those people who take care of many events at a particular time, assign the work in a team, and are also able to get the attendance.
